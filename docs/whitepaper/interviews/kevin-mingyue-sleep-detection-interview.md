# Outsider Interview — Sleep Detection

**Students:** Kevin Shi · Mingyue Zhou
**Project:** Sleep Detection
**Participant:** Maggie (friend)

---

**Me:** I'll just show you this first — don't think too much. What do you think this system is doing?

**Participant:** Umm… okay… there's a camera… so I guess it's watching people?

**Me:** Anything more specific?

**Participant:** Wait… is it trying to detect if someone is falling asleep?

**Me:** Yeah. So basically if the system thinks you're going fall asleep, it sends a notification to a nearby friend, and they wake you up. What do you think?

**Participant:** Ohh… okay, that's actually kind of interesting. It feels less aggressive than calling someone out.

**Me:** What was your first reaction to that?

**Participant:** That's kind of nice? Because I've definitely been in situations where I'm tired in class and don't want to be embarrassed.

**Me:** Would you prefer this over a teacher calling your name?

**Participant:** Yeah, for sure. If a teacher calls you out, that's way worse.

**Me:** Would you feel embarrassed if your friend got notified?

**Participant:** Hmm… a little bit? But not too bad. Like if it's someone I'm close to, it's fine. They'd probably just tap me and we'd both laugh.

**Me:** What kind of notification should the friend receive?

**Participant:** Something subtle. Like a small vibration or a message like "your friend might be falling asleep." Not anything loud or obvious.

**Me:** Does this system feel like it's helping or controlling behavior?

**Participant:** I think it's more on the helpful side. It doesn't feel like it's trying to discipline me, just kind of support me.

**Me:** Does it still feel like surveillance?

**Participant:** Yeah… a little. Because the camera is still watching you the whole time. Even if the response is private, that part doesn't really go away.

**Me:** Do you think this would actually help you stay engaged in class?

**Participant:** Yeah, I think so. Just knowing that someone might nudge me would probably make me try to stay awake more!

**Me:** Would you actually want to use this?

**Participant:** I think… maybe? If I trust the person next to me, then yeah. But if it's random seating, probably not.
