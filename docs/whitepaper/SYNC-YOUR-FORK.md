## Step 1: Sync your fork on GitHub

1. Go to **your fork** on github.com (it'll be something like `github.com/YOUR-USERNAME/smart-objects-cameras`)
2. You should see a banner near the top that says something like **"This branch is X commits behind kandizzy:main"**
3. Click the **Sync fork** button — a small dropdown will appear
4. Click **Update branch** in that dropdown

That's it for the web side. Your fork on GitHub now matches the class repo.

---

## Step 2: Pull the changes in GitHub Desktop

1. Open **GitHub Desktop**
2. Make sure the **smart-objects-cameras** repository is selected in the top-left dropdown
3. Click **Fetch origin** (top bar)
4. If it says there are changes to pull, click **Pull origin**

Your local files are now up to date.
